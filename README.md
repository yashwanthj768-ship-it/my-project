# Customer Churn Prediction

## Overview
This project predicts customer churn using machine learning techniques.
The goal is to help businesses identify customers likely to leave
and take preventive actions.

## Dataset
- Telco Customer Churn dataset
- Target variable: Churn (Yes/No)

## Tools Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
